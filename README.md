# Asteroid Miner
A VR implementation of O-mine board game

## A simple puzzle game
Use light probes to see where the valuable inclusions are hiding within the asteroid.

## VR target platforms.
This app needs neither networked nor cross-platform play. 
If we can build for different platforms, we'll be broadening our audience.

### HTC Vive XR Elite
Also, for Daniel's VR HMD, the HTC Vive XR Elite, need to add the Vive repo to Unity.
[InternetShortcut]
URL=https://www.mindport.co/vr-builder-tutorials/htc-vive-device-setup

Then fiddle with the target settings and get it to build. 
Would be simpler to set it up one way for PC VR through SteamVR, but it's just so easy to build for the wireless XR Elite HMD, copy over the .apk (android) binary, and test away. No wires, no WiFi, no hassles. 

### PC VR
Vive XR Elite can also be a wired or wireless HMD for SteamVR. 
Someone has two HMDs for PC VR, not sure if they're currently set up.

### Steam Frame
When the Deckard HMD, now known as the Steam Frame, releases, perhaps that will be a good target and it'll be worth getting our hands on one.

### Meta Quest
Also would like to test on Meta Quest 3 or similar, 

### Apple Vision Pro
The Cadillac of VR headsets in my opinion because of Apple's history of pioneering ease-of-use. Expensive, possibility of being discontinued, and additional learning. However, Unity does build for the Vision Pro, and its full face and eye tracking would add features to future projects.
